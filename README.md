## RPS

Welcome to the RPS (Rock Paper Scissor) game.


## Folder structure

The folder `src` contains the sources. The GUI.java contains the main function to run the program.

The folder `backend` contains the function to calculate the winner.

The folder `enumerators` contains the enumerator for the possible choices of the game.

The folder `frontend`contains the UI.


## Setting up your development environment

- Install Eclipse https://www.eclipse.org/downloads/packages/
- Install a Java Development Kit (JDK) (to be able to build without eclipse)
- Clone this repository and open it in Eclipse

## Build & run

Eclipse:

Click next to the play button and choose `RPS`. This builds and runs the project.

Command line:

- Open the command shell
- Go to the root of this repository
- Build the project with `javac RPS.java`
- Run the project with `java RPS`

## Run unit tests

Eclipse:

Click next to the play button and choose `CalculationTest`.
